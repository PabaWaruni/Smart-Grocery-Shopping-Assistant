# Smart-Grocery-Shopping-Assistant

A smart grocery list management application with category browsing, smart suggestions, and expiry reminders.

## Features

**Grocery List Management**
- Add items with name, unit, and optional category
- Remove items with one-click delete
- Persistent storage using JSON files

**Category Browsing**
- Pre-defined categories (Fruits & Vegetables, Dairy, Meat, etc.)
- Expand/collapse category items
- Click "Add" to pre-fill the manual add form with category and item name
- Edit and confirm before adding to list

**Smart Suggestions**
- **Missing Items**: Suggests items you purchased before (7+ days ago)
- **Healthier Alternatives**: Suggests healthier options (e.g., brown bread instead of white bread)
- Auto-updates every 30 seconds

**Expiry Reminders**
- View items expiring within 3 days
- Plan your shopping accordingly

**Purchase Management**
- Mark all items as purchased with one click
- Updates purchase history for future suggestions
- Clears the grocery list
